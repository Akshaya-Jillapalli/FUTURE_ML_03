About the Project

Intelligent Resume
Screening with NLP

This system helps recruiters automate candidate evaluation using Natural Language Processing. It extracts skills, computes similarity scores, and ranks applicants — reducing manual effort by over 80%.

Resume Parsing

Extract structured text from PDF and DOCX resumes automatically

Skill Extraction

Identify technical and soft skills using Named Entity Recognition

Similarity Scoring

TF-IDF + Cosine Similarity to compute resume–JD match score

Candidate Ranking

Sort all applicants by fit score and surface the best matches

Skill Gap Analysis

Pinpoint missing skills so recruiters know what to probe in interviews

Visual Reports

Dashboard-style analytics charts for hiring managers at a glance
